# Expense Tracker

A simple Expense Tracker built using HTML, CSS, and JavaScript.

## Features

* Add new income transactions

* Add new expense transactions

* Automatically calculate total balance

* Display transaction history

* Delete transactions

* Save transactions using Local Storage

* Automatically load transactions after page refresh

* Responsive design

## Technologies Used

* HTML

* CSS

* JavaScript

* Local Storage

## How to Use

1. Enter a transaction name.

2. Enter the transaction amount.

3. Select **Income** or **Expense**.

4. Click **Add Transaction**.

5. View the updated balance and transaction history.

6. Delete transactions when needed.

7. Transactions are automatically saved in the browser.
